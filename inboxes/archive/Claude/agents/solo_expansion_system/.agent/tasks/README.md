# Tasks

Thư mục này chứa các PRDs (Product Requirements Documents) và implementation plans cho mỗi feature/project.

## Cấu trúc

Mỗi task nên được tổ chức như sau:
- `project-name.md` - PRD cho project/feature
- `project-name-implementation.md` - Implementation plan

## Tạo Task mới

1. Tạo file PRD với project requirements
2. Tạo implementation plan với hướng dẫn từng bước
3. Link đến tài liệu liên quan trong `system/` và `sops/`
4. Cập nhật `readme.md` chính nếu cần
