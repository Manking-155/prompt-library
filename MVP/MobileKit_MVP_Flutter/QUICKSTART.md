# 🚀 Flutter MVP - Quick Start

## Setup (5 minutes)
1. Install Flutter SDK
2. Install: `npm install -g mobilekit-mvp-cli`
3. Create Firebase project
4. Run: `mkf-mvp init`
5. Run: `flutter run`

## First Feature (2 hours)
```bash
mkf-mvp feature add --name user_profile --type crud-firestore
```

## Deploy (30 minutes)
```bash
mkf-mvp deploy stores --track beta
```

**Ship iOS + Android in 2-4 weeks!** 🚀
