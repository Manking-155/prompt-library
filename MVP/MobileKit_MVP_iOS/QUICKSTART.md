# 🚀 iOS MVP - Quick Start

## Setup (5 minutes)
1. Install: `npm install -g mobilekit-mvp-cli`
2. Create Firebase project
3. Run: `mki-mvp init`
4. Open in Xcode and run!

## First Feature (2 hours)
```bash
mki-mvp feature add --name UserProfile --type swiftui-view
```

## Deploy (15 minutes)
```bash
mki-mvp deploy testflight
```

**Ship in 2-4 weeks!** 🚀
