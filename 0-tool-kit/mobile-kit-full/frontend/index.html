Getting Started with MobileKit

MobileKit is an AI-powered toolkit for mobile development that uses specialized agents to automate and enhance your development workflow.

Quick Start

  1. Create a new project using the Project Setup wizard
  2. Configure your agents in the Agents section
  3. Build workflows to automate your development process
  4. Use the Command Center for quick actions

Example: Creating a New Flutter Project

mk new-project --type flutter --name "MyApp" --package "com.example.myapp"
