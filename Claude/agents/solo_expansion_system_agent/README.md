### Vietnamese Tax System
- **Personal Income Tax**: 5-35% progressive rates
- **Value Added Tax (VAT)**: 10% standard rate
- **Business License Tax**: Annual requirement
- **Double Taxation**: 80+ country treaties
- **Compliance**: Quarterly and annual filing requirements

### Financial Planning
- **Tax Reserves**: 25-30% of income set aside
- **Emergency Fund**: 3-6 months expenses
- **Investment Fund**: 20% of profits
- **Business Expenses**: Target <$2,000/month
- **Profit Margins**: Target 60-80% on services
