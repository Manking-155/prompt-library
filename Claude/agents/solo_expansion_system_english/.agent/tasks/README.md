# Tasks

This directory contains PRDs (Product Requirements Documents) and implementation plans for each feature/project.

## Structure

Each task should be organized as:
- `project-name.md` - PRD for the project/feature
- `project-name-implementation.md` - Implementation plan

## Creating a New Task

1. Create a PRD file with project requirements
2. Create an implementation plan with step-by-step guide
3. Link to relevant documentation in `system/` and `sops/`
4. Update main `readme.md` if needed
